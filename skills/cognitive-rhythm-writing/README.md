# cognitive-rhythm-writing

説明的な文章に認知モードの切り替えと未回収の緊張を設計し、読み進めるリズムを作るための文章規範です。

このスキルは `japanese-tech-writing` を併用します。`SKILL.md` の相対参照を保つため、両方のスキルを同じ `skills/` 配下へ配置します。
